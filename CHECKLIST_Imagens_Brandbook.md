# ESPECIFICAÇÕES TÉCNICAS GERAIS

## FORMATOS VETORIAIS:
- **SVG:** Sempre que possível (escalável, web-friendly)
- **PDF:** Para gráficas (vetorial, CMYK)
- **EPS:** Se solicitado por fornecedor

## FORMATOS RASTER:
- **PNG:** Transparência, web, mockups
- **JPG:** Fundos sólidos, e-mail, impressão básica
- **Resolução mínima:** 300dpi para impressos, 72dpi web ok

## ESPAÇO DE COR:
- **RGB:** Digital (web, telas, Instagram)
- **CMYK:** Impressão offset/gráfica
- **Pantone:** Bordado, serigrafia, branding especial
